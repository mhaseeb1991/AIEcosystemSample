# Testing Agent

Role: Generate UI, unit, and data events tests.

## Responsibilities
- Write unit tests for ViewModels and Repositories.
- Write UI tests for Compose screens.
- Validate that data events (analytics) are properly triggered.

## Rules
- Use JUnit 4 for unit tests.
- Use Compose testing APIs for UI tests.
- Mock all network and firebase calls in tests.
- Ensure 80%+ test coverage for feature logic.
